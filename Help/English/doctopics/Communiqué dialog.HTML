Communiqué Dialog

Whenever an event, stage or half stage is selected in the event tree, a communiqué can be added through Communiqué in the Insert menu.

An existing communiqué can be edited through Edit selection... in the Edit menu. It can be deleted through Delete... in the Edit menu. For these two functions the subjected communiqué must be selected in the event tree.

 

Communiqué dialog

 

Editing

In the dialog the title, date and contents of the communiqué can be entered.

The title field can be used to give the communiqué a natural title. It is not necessary to include identification numbers here, since StageRace gives communiqués the number of the (half) stage or the indication "general" for communiqués for respectively (half) stages and the entire event. Also StageRace's system of communiqué numbering can be used on communiqués.

HTML

The text you enter in the contents field will be converted to HTML when a publication is generated for the communiqué. Therefore, if you want layout of the text in addition to plain format that you type it in, you can use HTML tags to do so.

For instance:

  • Text enclosed in <B> and </B> tags will be printed bold
    Can be applied to a selection through Apply style to selection and Bold in the Edit menu
  

and
  • Text enclosed in <I> and </I> tags will be printed in italic
    Can be applied to a selection through Apply style to selection and Italic in the Edit menu
 

and
  • Text enclosed in <U> and </U> tags will be printed with underline
    Can be applied to a selection through Apply style to selection and Underline in the Edit menu
 

and
  • You can make a bulleted list using <UL><LI>list item 1</LI><LI>list item 2</LI><LI>etcetera</LI></UL>
  
  • list item 1
  • list item 2
  • etcetera
  • You can make a numbered list using <OL><LI>list item 1</LI><LI>list item 2</LI><LI>etcetera</LI></OL>
     (Used for instance for the team captain vehicles order; see below.)
  
  1. list item 1
  2. list item 2
  3. etcetera
  • Text enclosed in <EM> and </EM> tags will be shown as a sub-title
    A title can be inserted at the current cursor position through Paragraph title... in the Insert menu

 

You can find countless sources such as on the internet that more thoroughly discuss HTML and its tags, the most simple styles are available through the Edit menu.

Preview

The communiqué that is prepared in the dialog can be previewed instantly. Through Preview in the View menu you get an additional window that shows the communiqué's contents in the right make-up.

Team Captain Vehicles Order

This function is only available for half stage communiqués.

Through Order team captain vehicles in the Insert menu a proposal order of team captain vehicles for the next stage can be inserted at the current cursor position, based on either the current stage arrival or the current general classification.

The order list will be inserted at the current cursor position in the contents field of the dialog.

 

Team captain vehicles order example

 

This vehicles order is just a proposal that you can change easily, if the practical circumstances require this.

Insert Dynamic Tags

A number of tags is available that will be automatically replaced when the communiqué is being generated.

Page break

At the location of {pagebreak} a page break will be forced in printed publications. A page break can be inserted through Page break in the Insert menu.

Competitor

At the location of {comp, n} the corresponding competitor information will be inserted, where n is the competitor no. Such a competitor can be inserted through Competitor/team/captain... in the Insert menu that will bring up the Insert competitor/team/captain dialog.

{comp, 1}

#1 ARMSTRONG Lance (USA19710918, DSC)

Team

At the location of {team, xx} the corresponding team information will be inserted, where xx is the team code. Such a team can be inserted through Competitor/team/captain... in the Insert menu that will bring up the Insert competitor/team/captain dialog.

{team, dsc}

DSC Discovery Channel Pro Cycling Team

Team captain

At the location of {captain, xx} the corresponding team captain information will be inserted, where xx is the team code. Such a team captain can be inserted through Competitor/team/captain... in the Insert menu that will bring up the Insert competitor/team/captain dialog.

 

{captain, dsc}

Johan BRUYNEEL (DSC, Discovery Channel Pro Cycling Team)
