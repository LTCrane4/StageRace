General Settings

Through Settings... in the Configure menu, and then choosing for the General tab, you can edit the general settings for the event.

When the small size of the settings control holds you back from properly making the settings, you can maximize the control in which you make these settings, by double-clicking on any whitespace in the control.

 

 

The following general settings can be made.

Main Properties

Enter the following informative fields:

  • Organiser name;
  • Event title;
  • Event subtitle;
  • Description or actual name of the main location;
  • Dates of the event; possibly the actual dates, but maybe better the name of a holiday weekend; and
  • Main competitor category, such as "Juniors" or "GSI/II".

Classification Sponsors

The sponsors of classifications can be entered for all types of classifications and for all the masks. Classification sponsors are printed in addition to the event title and sub title.

Masks

You can enable the masks that are actually used for the event. Also you choose if the mask numbers ("Mask #") should be printed on publications.

You can enter the names and the classification sponsors for each mask.

Miscellaneous

You can switch the input accuracy for arrival dialogs between 1/100ths and 1/1000ths of a second.

You can set the event to begin with a prologue.

You can choose that half stage numbers are suppressed when a full stage or day only has a single half stage.
