MultiLanguage Strings

The following language strings are used so far in the MultiLanguage2005.mdb database.

 

For the less self-explanatory strings a explanation and/or value is given. The more self-explanatory, more generic strings are grouped in categories, just like all the others, but are occasionally used for other functions than their grouping implies. This can be done, since their value is basically equal to the identifier name. (For instance the identifier Date will have a value "Date" in English.)
