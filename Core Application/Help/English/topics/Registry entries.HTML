Registry Entries

The following system registry keys are in use by StageRace.

Note that making changes to registry keys manually should be left to very experienced computer users.

Root Keys

In the root section of the registry StageRace stores values with the purpose of:

  • Registering the StageRace .str file type extension to the operating system; and
  • Registering the StageRace Server interfaces, coclasses, inproc-server and type library to the operating system.

Current User Keys

In the user section of the registry StageRace stores values with the purpose of:

  • Storing user preferences on the registration of the StageRace .str file type extension; and
  • Storing user preferences for the operation of StageRace (typically options). These are stored in base key HKEY_CURRENT_USER\Software\Totaal Software\StageRace, with the following values:

Value name

Value data

File dialogs

Remembers the last folder visited for each type of file dialog

Import

Remembers the import file characteristics that are chosen in the Import classification file dialog and the Control import classification dialog

ListCompetitors

Remembers order and widths of the columns in the competitors view

ListTeams

Remembers order and widths of the columns in the teams view

MainFrame

Remembers the positioning of the main frame

Options

Remembers the options that you choose through the Options dialog

Recent File List

Remembers the most recently opened files (for the File menu)

Settings

Unused

Time trial arrival

Remembers to anticipate the arrival time and which time to anticipate

Tip of the Day

Remembers if you want Tip of the Day displayed on start up and also which tip was shown last

WindowPositions

Remembers the positioning of various dialogs

Local Machine Keys

In the local machine section of the registry StageRace stores values with the purpose of:

  • Registering the StageRace .str file type extension to the operating system;
  • Registering the StageRace Server interfaces, coclasses, inproc-server and type library to the operating system; and
  • Storing system preferences for the operation of StageRace (typically options). These are stored in base key HKEY_LOCAL_MACHINE\Software\Totaal Software\StageRace, with the following values:

Value name

Value data

Export paths

Remembers the paths that you set in the Options dialog for exporting files

Windows® will cache connection information of the StageRace Server in this section of the registry.

Uninstall Information

Base keys HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall\StageRace 2005 and HKEY_CURRENT_USER\UserUninstall\StageRace 2005 hold uninstall information of StageRace.

Removing Registry Keys

You can remove all involved registry keys of StageRace by running StageRace.exe with the unreg parameter/switch.

You can for instance do this through Run in the Start menu of Windows®, after which you type the following command:

C:\Program Files\StageRace 2005\StageRace.exe unreg

Activate OK once you are done typing.

Note that the exact line to type might be different, depending on the folder that you chose during the program installation.

Also note that this feature exists to support manual installations only. Therefore the uninstall information registry key and values are not removed by it.
