Publications

Elements

All the publications that are generated by StageRace have a number of common elements.

These elements can be divided into:

  • Header
  • Columns
  • Footer

Using the Page Setup dialog you can configure additional headers and footers, such as page numbering.

Also a standardized way of naming publication files is applied.

Available Publications

The following publications are generated by StageRace:

  • Table of contents;
  • Stage classification;
  • Arrivals list for time trials on gross time;
  • General classification;
  • General classification sorted on competitor no.;
  • Masks:
    • Stage classifications; and
    • General classifications;
  • Teams classification;
  • Sprint classification;
  • Points classification;
  • Climbing classification;
  • Most aggressive rider classification;
  • Time trial starting order list;
  • Cross out list;
  • Sign in list;
  • Competitors list;
  • Competitors list per team; and
  • Communiqués for the event, for full stages or days, and for half stages.

You can select the language that is used for publications through the publication settings.

Publications can be generated for multiple profiles.

Publications support Unicode.
